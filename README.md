# AutoMatch Car Recommender

AutoMatch is a machine learning-based car recommendation system that helps users identify suitable vehicles based on preferences such as budget, fuel efficiency, features, and practical needs.

## Tech Stack
- Python
- Streamlit
- Machine Learning
- Pandas
- React / Vite
- JavaScript
- Tailwind CSS

## Features
- Personalized car recommendations
- Preference-based vehicle matching
- Inventory-driven suggestions
- Frontend and backend project structure
- Data collection pipeline for car listings

## Project Structure
- `app.py` – main application entry point
- `backend/` – recommendation logic and backend components
- `frontend/` – user interface
- `Inventory/` – vehicle dataset files
- `scrape_honda.py` – data scraping script

## Purpose
This project was built to explore how machine learning and software engineering can be combined to create practical, user-focused recommendation systems.
