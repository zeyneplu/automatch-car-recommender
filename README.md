# AutoMatch Car Recommender

Preference-based Honda vehicle recommendation system built with React, FastAPI, and Streamlit
Car recommendation system for Honda inventory using weighted matching and full-stack app components
## Project Structure
- `app.py` – main application entry point
- `backend/` – recommendation logic and backend components
- `frontend/` – user interface
- `Inventory/` – vehicle dataset files
- `scrape_honda.py` – data scraping script
